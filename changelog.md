# Changelog

## v0.1.0 - 2025-12-22

### Added
- CSV preview endpoint and UI preview table.
- KML generation for Points with optional styling (icon URL, color, scale).
- KML generation for Links (LineString) with line styling (color, width).
- Graph mode to generate Points + Links in a single KML with point deduplication.
- Full-stack Docker setup (docker-compose) for easy startup.
- Documentation: example CSV files and UI screenshots.
- GitHub Actions CI: backend tests (pytest) and frontend build check.
