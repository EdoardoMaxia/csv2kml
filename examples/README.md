# Examples

This folder contains sample CSV files you can use to try csv2kml quickly.

## Files

- `points_simple.csv`
  - Use with the **Points** tab.
  - Columns: `name`, `latitude`, `longitude`, plus extra fields for description.

- `links_simple.csv`
  - Use with the **Links** tab.
  - Columns for endpoint A and B (`a_lat`, `a_lon`, `b_lat`, `b_lon`) + metadata.

- `graph_points_and_links.csv`
  - Use with the **Graph** tab to generate points + links in a single KML.
